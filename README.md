# dext-system-plugin

> [Dext](https://github.com/vutran/dext) plugin to display system and battery info.

## Install

```bash
$ dpm install dext-system-plugin
```

*Requires [Dext](https://github.com/vutran/dext).*

## Usage

In Dext, type `system` or `battery`.
